# 📝 나의 할 일 목록 (My Todo App)

간단하고 직관적인 할 일 관리 웹 애플리케이션입니다.

## 주요 기능

- ✅ **할 일 추가/삭제**: 간편하게 할 일을 추가하고 삭제할 수 있습니다
- ✅ **완료 상태 관리**: 체크박스로 할 일의 완료 여부를 표시합니다
- ✅ **중복 방지**: 동일한 할 일이 중복으로 등록되지 않습니다
- ✅ **필터링**: 전체/진행중/완료 항목을 필터링하여 볼 수 있습니다
- ✅ **통계 표시**: 전체 할 일 개수와 완료된 할 일 개수를 실시간으로 확인할 수 있습니다
- ✅ **일괄 삭제**: 완료된 항목을 한 번에 삭제할 수 있습니다
- ✅ **로컬 저장**: LocalStorage를 사용하여 데이터를 브라우저에 저장합니다

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인 및 스타일링
- **JavaScript (ES6+)**: Vanilla JavaScript를 사용한 동적 기능 구현

2. 브라우저에서 `index.html` 파일 열기

   - 브라우저에 파일을 드래그 앤 드롭

별도의 서버나 빌드 과정이 필요하지 않습니다.

## 사용 방법

1. **할 일 추가**: 입력창에 할 일을 입력하고 "추가" 버튼을 클릭하거나 Enter 키를 누릅니다
2. **완료 표시**: 체크박스를 클릭하여 할 일의 완료 상태를 변경합니다
3. **삭제**: 각 항목의 "삭제" 버튼을 클릭하여 개별 삭제합니다
4. **필터링**: "전체", "진행중", "완료" 버튼을 클릭하여 원하는 항목만 볼 수 있습니다
5. **일괄 삭제**: "완료된 항목 삭제" 버튼을 클릭하여 완료된 모든 항목을 한 번에 삭제합니다

## 주요 코드 구조

### 데이터 구조
```javascript
{
  id: Number,        // 고유 식별자 (타임스탬프)
  text: String,      // 할 일 내용
  completed: Boolean // 완료 여부
}
```

### 주요 함수
- `addTodo()`: 할 일 추가
- `deleteTodo(id)`: 할 일 삭제
- `toggleTodo(id)`: 완료 상태 토글
- `setFilter(filter)`: 필터 설정
- `clearCompleted()`: 완료 항목 일괄 삭제
- `renderTodos()`: 화면 렌더링
